nota: shift es la fleacha hacia arriba.

DEV TOOLS
ctrl + shift + i : ver devtools
ctrl + shift + supr : borrar historial navegacion


WINDOWS
alt + esc: ver ventana anterior
alt + tab: cambiar a ventanas recientes.
win + d: desktop
win + e: ver carpetas en local
win + i: ver configuracion sistema
win + shift + s : captura pantalla con recorte

INTELLIJ
ctrl + h : ver herencia
ctrl + d: repetir linea
shift + f6: renombrar (refractor)
ctrl + shift + f10: correr programa actual.
ctrl + /: comentario una linea
ctrl + shift + / : comentario bloque
